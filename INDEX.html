Como Ativar o Projeto
Passos para conectar ao Supabase e rodar em produção
1

Criar projeto Supabase

Acesse supabase.com → New Project. Anote a URL e a Anon Key em Settings → API.

2

Preencher .env.local

Edite o arquivo .env.local na raiz do projeto com sua URL e chave.

3

Rodar as migrations SQL

No painel Supabase → SQL Editor → cole e execute os 4 arquivos da pasta supabase/migrations/ em ordem (0001 → 0004).

4

Iniciar o servidor de desenvolvimento

Na pasta do projeto, execute: npm run dev — depois acesse http://localhost:3000

5

Tornar-se admin

Registre sua conta, depois no Supabase → Table Editor → profiles → altere seu role de student para admin.
